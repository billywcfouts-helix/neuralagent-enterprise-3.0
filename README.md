Key features:
- Auto .env creation on first run
- OCR-powered visual anchoring in replays
- Full Watch & Learn recording (pynput)
- Specialized agents (Research, Code, Growth, Browser)
- Playwright hybrid web automation
- Live real/simulated mode switching
- Modern Streamlit GUI

Quick start: unzip, pip install -r requirements.txt, streamlit run gui/app.py
