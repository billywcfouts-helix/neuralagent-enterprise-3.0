# NeuralAgent Enterprise Automation 3.0

**Grok-Powered Local Desktop AI Agent System**

Full source code and complete project available here:

https://drive.google.com/drive/folders/1l6DUqxxWHSAu0rK8irrj_NdD47406Ugf

### Key Features
- Auto .env creation on first run
- OCR-powered visual anchoring in replays (much more robust)
- Full Watch & Learn recording using real pynput listeners
- Specialized agents: Research, Code, Growth, Browser (with Playwright hybrid)
- Live toggle between Real and Simulated computer control
- Modern beautiful Streamlit GUI
- Enterprise safety, logging, and Fast Replays

### Quick Start

```bash
# After downloading the full project from Google Drive
cd neuralagent-enterprise-automation
pip install -r requirements.txt
streamlit run gui/app.py
```

On first run, `.env` is created automatically.

See the Google Drive folder for the complete codebase including all Python files, build script, and documentation.
